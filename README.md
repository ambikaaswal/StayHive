# StayHive
StayHive is a full-stack web application that allows users to discover, list, and manage homestays across various locations. Built with Node.js, Express.js, MongoDB, and EJS, the platform supports secure user authentication, image uploads via Cloudinary, and dynamic search functionality—including voice search powered by the Web Speech API. Hosts can create and edit listings, while guests can browse stays using intuitive filters and responsive design. The project emphasizes modular architecture, accessibility, and clean UI, making it a scalable foundation for future features like chatbots, admin dashboards, and personalized recommendations.
